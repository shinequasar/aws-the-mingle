# User Personas

## Persona 1: 고객 (Customer)
- **이름**: MZ세대 헌팅포차 이용자
- **연령대**: 20~30대
- **특징**: 새로운 사람과의 만남에 관심, 디지털 기기에 익숙, 재미있는 경험 추구
- **목표**: 편리하게 주문하고, 다른 룸과 소통하며, 합석을 통해 새로운 인연 만들기
- **Pain Point**: 직원 호출 대기, 다른 룸과 소통할 방법 부재, 어색한 합석 분위기

## Persona 2: 관리자 (Admin)
- **이름**: 매장 운영자/직원
- **특징**: 다수의 룸을 동시 관리, 빠른 주문 처리 필요, 매장 운영 효율 중시
- **목표**: 실시간 주문 파악, 룸 상태 관리, 메뉴/매장 운영 효율화
- **Pain Point**: 주문 누락, 룸 이용 시간 관리 어려움, 부적절 행위 대응
