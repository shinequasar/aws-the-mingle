# NFR Requirements - Customer App

## 성능 요구사항

| 항목 | 요구사항 |
|------|---------|
| 초기 로딩 | < 3초 (태블릿 기준) |
| 페이지 전환 | < 300ms (SPA 클라이언트 라우팅) |
| 메뉴 이미지 | lazy loading 적용 |
| WebSocket 재연결 | 연결 끊김 시 자동 재연결 (3초 간격) |

## 사용성 요구사항

| 항목 | 요구사항 |
|------|---------|
| 터치 타겟 | 최소 44x44px |
| 반응형 | 태블릿 최적화 (768px~1024px) |
| 오프라인 | 장바구니 localStorage 유지 |
| 피드백 | 모든 액션에 로딩/성공/에러 피드백 |

## 보안 요구사항

| 항목 | 요구사항 |
|------|---------|
| 토큰 저장 | localStorage (PoC 수준) |
| API 호출 | Authorization Bearer 헤더 |
| 토큰 만료 | 401 응답 시 자동 로그아웃 |
