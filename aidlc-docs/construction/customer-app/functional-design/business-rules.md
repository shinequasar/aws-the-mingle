# Business Rules - Customer App (Frontend)

## BR-FE-01: 인증 흐름
- JWT 토큰 없으면 `/login`으로 리다이렉트
- 토큰 만료(16시간) 시 자동 로그아웃 → `/login`
- 모든 API 요청에 Authorization: Bearer {token} 헤더

## BR-FE-02: 장바구니
- localStorage에 저장 (새로고침 유지)
- 수량 최소 1, 최대 99
- 동일 메뉴 추가 시 수량 합산
- 총 금액 = Σ(단가 × 수량)
- 주문 성공 시 장바구니 비우기

## BR-FE-03: 주문 후 흐름
- 주문 성공 → 주문번호 표시 → 5초 카운트다운 → 메뉴 화면 자동 이동
- 주문 실패 → 에러 메시지 표시, 장바구니 유지

## BR-FE-04: WebSocket 알림 처리
- 연결: 로그인 성공 시 /ws 연결, /topic/room/{roomId}/notifications 구독
- 음료/합석 수신 → 수락/거절 액션 버튼 포함 토스트
- 메시지/사진 수신 → 확인 토스트
- 합석 성공 → 게임 페이지 자동 노출 안내 (이스터에그)

## BR-FE-05: 타이머
- 세션 시작 시 서버에서 expiresAt 조회
- 클라이언트 측 1분 간격 카운트다운
- 30분 이하 → 빨간색 경고
- 0분 → 시간 초과 알림

## BR-FE-06: 직원 호출
- 사유 목록: 물, 잔, 휴지, 젓가락, 숟가락, 에어컨, 앞접시
- 호출 후 "호출 완료" 피드백 (2초)

## BR-FE-07: 게임 접근
- 합석 성공 시 자동 노출 (NotificationToast에서 게임 페이지 링크)
- 하단 네비게이션에서도 직접 접근 가능
