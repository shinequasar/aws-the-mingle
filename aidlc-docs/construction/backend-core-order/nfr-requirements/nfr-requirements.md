# NFR Requirements - Backend Core Order

## 성능 요구사항

| 항목 | 요구사항 |
|------|---------|
| 동시 접속 | 매장당 룸 패드 10~30대 + 관리자 1대 |
| API 응답 시간 | 메뉴 조회 < 1초, 주문 생성 < 2초 |
| SSE 이벤트 전달 | 주문/호출 알림 2초 이내 |
| DB 커넥션 풀 | 최대 20개 (HikariCP) |

## 보안 요구사항

| 항목 | 요구사항 |
|------|---------|
| 인증 | JWT 토큰 (16시간 만료) |
| 비밀번호 | bcrypt 해싱 |
| 로그인 제한 | 5회 실패 시 15분 잠금 |
| API 보호 | JWT 필터로 모든 API 보호 (로그인 제외) |

## 가용성/안정성

| 항목 | 요구사항 |
|------|---------|
| 배포 환경 | 로컬 서버 단일 인스턴스 |
| 데이터 백업 | MySQL 일일 백업 권장 |
| 에러 처리 | 글로벌 예외 핸들러, 사용자 친화적 에러 응답 |
| 로깅 | SLF4J + Logback, 파일 로깅 |
